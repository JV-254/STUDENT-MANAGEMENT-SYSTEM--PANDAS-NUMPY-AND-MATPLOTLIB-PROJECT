# STUDENT-MANAGEMENT-SYSTEM--PANDAS-NUMPY-AND-MATPLOTLIB-PROJECT
##  Project Overview

The Student Performance Dashboard is a data analysis project built using **Python**, **NumPy**, **Pandas**, and **Matplotlib**. The project analyzes students' academic performance by calculating key performance metrics, generating insights, and visualizing the results through charts.

This project demonstrates the complete data analysis workflow, from loading and cleaning data to feature engineering, statistical analysis, and data visualization.

---

##  Objectives

* Load and explore student performance data.
* Clean and validate the dataset.
* Calculate total marks and average scores.
* Assign grades based on average scores.
* Determine pass/fail status.
* Perform statistical analysis on student performance.
* Visualize performance using various charts.
* Export the processed dataset for further use.

##  Dataset

The dataset contains student academic records with the following fields:

* Student ID
* Name
* Math Score
* English Score
* Science Score
* Attendance (%)

Additional columns created during analysis include:

* Total Marks
* Average Score
* Grade
* Status (Pass/Fail)

##  How to Run

1. Clone the repository.

```bash
git clone https://github.com/JV-254/STUDENT-MANAGEMENT-SYSTEM--PANDAS-NUMPY-AND-MATPLOTLIB-PROJECT
```

2. Open the project folder.

3. Install the required libraries.

```bash
pip install pandas numpy matplotlib openpyxl
```

4. Launch Jupyter Notebook.

```bash
jupyter notebook
```

5. Open the notebook and run all cells.
